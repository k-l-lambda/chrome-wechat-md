# 第三方库

需要下载以下库文件到此目录:

## 1. marked.js (Markdown 解析器)

**下载地址**: https://cdn.jsdelivr.net/npm/marked@11.1.1/marked.min.js

**或使用国内镜像**:
```bash
curl -o marked.min.js https://unpkg.com/marked@11.1.1/marked.min.js
```

## 2. juice.js (CSS 内联工具)

**下载地址**: https://cdn.jsdelivr.net/npm/juice@9.1.0/dist/juice.min.js

**或使用国内镜像**:
```bash
curl -o juice.browser.min.js https://unpkg.com/juice@9.1.0/dist/juice.min.js
```

## 手动下载步骤

1. 访问上述 CDN 链接
2. 复制全部内容
3. 保存为对应文件名到此目录

## 验证

文件下载完成后,此目录应包含:
- `marked.min.js` (~50KB)
- `juice.browser.min.js` (~100KB)
